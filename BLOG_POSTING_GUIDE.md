## How to Post a Blog (2 Minutes)

### Step 1: Access Admin Panel
1. Go to: `https://yoursite.netlify.app/admin`
2. Click "Login with GitHub"
3. Authorize (first time only)

### Step 2: Create New Post
1. Click **"Blog Posts"** in sidebar
2. Click **"New Blog Post"** button

### Step 3: Fill in Details
- **Title:** Your post title
- **Date:** Auto-filled (can change if needed)
- **Category:** 
  - Development Update
  - Technical Deep Dive
  - Founder Journey
  - Industry Insights
- **Read Time:** "5 min read" (estimate)
- **Description:** 2-3 sentence preview for blog cards
- **Body:** Write your full post here

### Step 4: Publish
1. Toggle **"Published"** to ON (if you want it live)
2. Click **"Publish"** button (top right)
3. Done! Post is live in 30 seconds! ğŸ‰

## ğŸš¨ Common Issues

**Problem: Changes not showing**
- Hard refresh: Ctrl+F5 (Windows) or Cmd+Shift+R (Mac)
- Clear browser cache
- Wait 30-60 seconds for deployment

**Problem: Formatting looks wrong**
- Check markdown syntax
- Preview before publishing
- Use toolbar instead of manual markdown

**Problem: Can't upload image**
- Check file size (under 5MB)
- Use JPG or PNG format
- Make sure `images/uploads` folder exists

**Problem: Post not appearing on blog page**
- Currently: Blog page has static posts
- Solution: I can update blog.html to load dynamically
- Ask me to set this up!
